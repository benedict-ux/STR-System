# Tech Stack

- Pure HTML, CSS, and vanilla JavaScript — no frameworks, no build tools, no dependencies
- Single file: `index.html` contains all markup, styles, and scripts inline
- Runs directly in any browser (open file locally, no server required)

## Conventions

- Inline `<style>` block in `<head>` for all CSS
- Inline `<script>` block at end of `<body>` for all JS
- No external libraries, CDNs, or package managers
- No transpilation or bundling step needed
